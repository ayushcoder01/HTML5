HTML Tables :-
==============
-> Purpose:
 - HTML tables are used to organize and present data in rows and columns.
 - Ideal for tabular data: pricing, schedules, product listings, reports.

-> Basic Structure
   <table>
     <tr>         <!-- Table Row -->
       <th>       <!-- Table Header (bold & centered) -->
       <td>       <!-- Table Data Cell -->
   </table>

-> Example:
   <table>
     <tr>
       <th>Name</th>
       <th>Age</th>
     </tr>
     <tr>
       <td>Ayush</td>
       <td>22</td>
     </tr>
   </table>

-> Important Tags
 - <table>:
   • The main table wrapper.

 - <tr> (Table Row):
   • Represents a row in the table.

 - <th> (Table Header):
   • Bold, centered header cell.

 - <td> (Table Data):
   • Standard data cell.

-> Advanced Structure (Semantic Elements)
 - <thead>:
   • Groups all header rows.
   • Helps accessibility and SEO.

 - <tbody>:
   • Contains the body rows (main content).

 - <tfoot>:
   • Used for footers: totals, summaries, etc.

-> Example:
   <table>
     <thead>  
      <tr><th>Item</th><th>Price</th></tr>
     </thead>
     <tbody>
       <tr><td>Book</td><td>₹250</td></tr>
       <tr><td>Pen</td><td>₹20</td></tr>
     </tbody>
     <tfoot>
       <tr><td>Total</td><td>₹270</td></tr>
     </tfoot>
   </table>

-> Useful Table Attributes
 - border:
   • Adds border to table and cells
   • Example: <table border="1">

 - colspan:
   • Merges cells horizontally across columns
   • Example: <td colspan="2">Merged</td>

 - rowspan:
   • Merges cells vertically across rows
   • Example: <td rowspan="2">Merged</td>

-> Accessibility Best Practices
 - Use <caption>:
   • Describes the table content
   • Example: <caption>User Data Table</caption>

 - Use scope in <th>:
   • <th scope="col"> → for column headers
   • <th scope="row"> → for row headers

 - Structure matters:
   • <thead>, <tbody>, and <tfoot> help screen readers
   • Keep markup semantic and meaningful

-> Summary
 - Use tables only for data (not layout!)
 - Maintain semantic structure with thead/tbody/tfoot
 - Ensure accessibility using <caption> and scope attributes
