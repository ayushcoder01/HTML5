HTML TABLES :-
==============
-> Overview:
 - Tables are used to display data in rows and columns.
 - Built using the <table> tag with nested elements for headers, rows, and cells.

-> Core Tags:
 - <table> : Defines the table container.
 - <tr> : Table row (table row).
 - <th> : Table header cell (bold + centered by default).
 - <td> : Table data cell.

-> Optional Tags:
 - <caption> : Title for the table.
 - <thead> : Groups header rows.
 - <tbody> : Groups body rows.
 - <tfoot> : Groups footer rows.

 - Syntax
   <table>
     <tr>
       <th>Header 1</th>
       <th>Header 2</th>
     </tr>
     <tr>
       <td>Data 1</td>
       <td>Data 2</td>
     </tr>
   </table>

-> Example:
   <table border="1">
     <caption>Student Marks</caption>
     <thead>
       <tr>
         <th>Name</th>
         <th>Math</th>
         <th>Science</th>
       </tr>
     </thead>
     <tbody>
       <tr>   
         <td>John</td>
         <td>80</td>
         <td>90</td>
       </tr>
       <tr>
         <td>Jane</td>
         <td>85</td>
         <td>88</td>
       </tr>
     </tbody>
   </table>

-> Attributes:
 - border: Adds border to table.
 - rowspan: Cell spans multiple rows.
 - colspan: Cell spans multiple columns.
 - width & height: Dimension of table or cells.

-> Best Practices:
 - Use <thead>, <tbody>, <tfoot> to separate structure from styling.
 - Avoid using tables for page layout (use CSS grid/flex instead).
 - Provide <caption> and <th> for accessibility.
 - Keep table layouts simple and readable.

-> Expert Tips:
 - Style tables using CSS to add zebra-stripes, hover effects, and borders.
 - Combine colspan and rowspan carefully to avoid broken structures.
 - Make tables responsive using overflow scroll or display blocks.
