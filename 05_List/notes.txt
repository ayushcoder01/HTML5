HTML LIST :-
============
-> Overview:
 - Lists are used to group related items in a structured way.
 - HTML supports three main types of lists: Ordered, Unordered, and Description lists.

-> Ordered List (<ol>):
 - Displays items in a numbered sequence.
 - Each list item is wrapped inside <li> (list item) tag.
 - Numbers can be customized using the "type" attribute (1, A, a, I, i).

 - Syntax
   <ol>
     <li>Item 1</li>
     <li>Item 2</li>
   </ol>

-> Example:
   <ol type="A">
     <li>HTML</li>
     <li>CSS</li>
     <li>JavaScript</li>
   </ol>

-> Unordered List (<ul>):
 - Displays items with bullet points.
 - Bullet style can be adjusted using the "type" attribute (disc, circle, square).

 - Syntax
   <ul>
     <li>Apple</li>
     <li>Banana</li>
   </ul>

-> Example:
   <ul type="square">
     <li>Pen</li>
     <li>Pencil</li>
     <li>Notebook</li>
   </ul>

-> Description List (<dl>):
 - Used to display terms and definitions.
 - Consists of <dt> (definition term) and <dd> (definition description).

 - Syntax
   <dl>
     <dt>HTML</dt>
     <dd>A markup language for web pages</dd>
   </dl>

-> Example:
   <dl>
     <dt>CPU</dt>
     <dd>Central Processing Unit</dd>
     <dt>RAM</dt>
     <dd>Random Access Memory</dd>
   </dl>

-> Best Practices:
 - Keep list items short and to the point.
 - Use ordered lists when order/sequence matters.
 - Use unordered lists for simple bullet-style lists.
 - Avoid nesting too many levels deep unless necessary.

-> Expert Tips:
 - Lists are great for navigation menus and can be styled using CSS.
 - Combine <ul> with <nav> for semantic menus.
 - Use CSS to convert lists into horizontal menus or cards by removing default bullets.
