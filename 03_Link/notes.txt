HTML HYPERLINKS :-
==================
-> Overview:
 - Hyperlinks allow users to navigate from one web page to another.
 - Created using the <a> (anchor) tag with the href attribute.
 - Can link to external websites, internal pages, email addresses, or files.

 - Syntax
   <a href="URL">Clickable Text</a>

-> Types of Links:
 - Absolute URL: Full web address (https://example.com)
 - Relative URL: Link within the same site (contact.html)
 - Email Link: href="mailto:example@gmail.com"
 - Telephone Link: href="tel:+911234567890"
 - Target Attribute:
   • _self  (default) — opens in same tab
   • _blank — opens in new tab

-> Example:
   <a href="https://google.com">Visit Google</a>
   <a href="about.html">About Us</a>
   <a href="mailto:info@example.com">Send Email</a>
   <a href="tel:+919876543210">Call Us</a>
   <a href="https://linkedin.com" target="_blank">Open LinkedIn in New Tab</a>

-> Best Practices:
 - Use descriptive link text (avoid “click here”).
 - Use target="_blank" only when necessary and add rel="noopener noreferrer".
 - Style links via CSS instead of inline.
 - Use title attribute to provide extra information for users and screen readers.

-> Expert Tips:
 - Combine hyperlinks with semantic tags like <nav> and <footer> for better structure.
 - Use HTTPS links only (secure).
 - Use anchor tags for file downloads as well, e.g., <a href="resume.pdf" download>.


PATH TYPES IN HTML :-
=====================
-> Overview:
 - Paths (or URLs) are used in attributes like href and src to locate files.
 - Two main types: Absolute Path and Relative Path.

-> Absolute Path:
 - Full path to the resource including protocol and domain.
 - Always starts with http://, https://, or file://
 - Used to link external sites/resources.

 - Syntax
   <img src="https://example.com/images/logo.png">

-> Relative Path:
 - Used to link files within the current project folder.
 - Depends on the current file location.

 • Same Folder:
   Syntax: <img src="photo.jpg">
 • Inside Sub-folder:
   Syntax: <img src="images/photo.jpg">
 • Go Back a Folder:
   Syntax: <img src="../photo.jpg">

-> Example Folder Structure:
   project/
    ┣ index.html
    ┣ about.html
    ┗ images/
       ┗ logo.png

-> Example Usage:
   <img src="images/logo.png">     (relative)
   <img src="https://site.com/assets/img/logo.png">  (absolute)

-> Best Practices:
 - Use relative paths for internal files to keep the site portable.
 - Use absolute paths for external resources or CDN links.
 - Avoid spaces in folder and file names (use hyphens or underscores).

-> Expert Tips:
 - Leading slash “/” makes it an absolute path within the site domain: /assets/img/logo.png
 - Use relative paths in development and convert to absolute in production when needed.
